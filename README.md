# SEO-Agent-Bot

SEO Agent Bot

This project is a simple AI tool that helps generate SEO ideas.

You enter:

a website domain

a topic

The app uses AI to:

generate 20 keyword ideas

create a structured content brief

The results are saved and shown so they can be used again later.




Tech Used

Next.js – for the website

LangChain + OpenAI – for AI gen

Prisma + SQLite – to save it

GitHub – for version control and collab




How to Run It Locally
npm install
npm run dev


then open up the local host website listed
